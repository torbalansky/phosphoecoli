Site Map


Homepage

The Homepage serves as the entry point to the application, offering users an intuitive search function for quick access. Users select their search criteria and input the corresponding search term.


Search Page

The Search Page hosts the protein list containing all phosphoproteins and their respective phospho-sites, along with a search engine. Users can search for phospho-sites by UniProt code, gene, or protein name. The search functionality is sensitive and requires users to input their query correctly. For example, if a user wants to search by UniProt ID, they must enter only the UniProt code. Additionally, users can filter their search results based on the phosphorylated amino acid, using the dropdown menu.


Users can use the "Clear" button provided in the form to return to the full list of proteins if they wish to start a new search or explore all proteins again.


In the detailed view of a protein entry in PEDB, users are presented with a comprehensive overview. At the forefront, users immediately encounter the exact phospho-site they clicked on, ensuring precision and clarity in their exploration. Following this, a structured list unveils all phospho-sites associated with the protein. Through the interactive chart, users can focus on specific regions of interest, facilitating a nuanced understanding of phospho-site distribution. Below this visualization, the protein sequence unfolds, with phospho-sites prominently highlighted. This intuitive presentation enables users to pinpoint phospho-sites within the broader protein sequence.

Additionally, users benefit from convenient access to valuable resources via curated links. While PEDB serves as an invaluable repository for phospho-site data, it is essential to underscore that its purpose is complementary, not substitutive, to established databases such as UniProt, NCBI, PDB etc. Moreover, users have the option to download a PDF file containing the data related to this entry, enabling offline access and further analysis.


Contribute

The Contribute page allows users to submit new phosphoprotein/phosphosite data to our database. Access to the submission form requires registration, and all submissions undergo thorough review before approval, including user registrations. Permission-based access is enforced.


Resources

This page offers a variety of valuable resources related to protein phosphorylation.


Contact

Users can provide feedback or make inquiries through this interface.


About/General Information/Citation

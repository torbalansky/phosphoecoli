Site Map


Homepage

The Homepage serves as the entry point to the application, offering users an intuitive search function for quick access. Users select their search criteria and input the corresponding search term.


Search Page

The Search Page hosts the protein list containing all phosphoproteins and their respective phosphosites, along with a search engine. Users can search for phosphosites by UniProt code, gene, or protein name. The search functionality is sensitive and requires users to input their query correctly. For example, if a user wants to search by UniProt ID, they must enter only the UniProt code. Mixing protein/gene names with UniProt codes may result in inaccurate search results. Additionally, users can filter their search results based on the phosphorylated amino acid, including H (Histidine), Y (Tyrosine), T (Threonine), or S (Serine), using the dropdown menu.


After performing a search, the search form is automatically cleared. However, users can use the "Clear" button provided in the form to return to the full list of proteins if they wish to start a new search or explore all proteins again.


Detailed information and features such as PDF downloads are accessible via the view button for each phosphoprotein.


Contribute

The Contribute page allows users to submit new phosphoprotein/phosphosite data to our database. Access to the submission form requires registration, and all submissions undergo thorough review before approval, including user registrations. Permission-based access is enforced.


Resources

This page offers a variety of valuable resources related to protein phosphorylation.


Contact

Users can provide feedback or make inquiries through this interface.
